By: Adam Lieberman



Introduction

There are tons of machine learning algorithms out there and a lot of them were built to accomplish the same goals. Here, we will discuss some popular machine learning algorithms, provide a high-level overview of them, when to use the algorithm, and some pros and cons.

Understanding Your Data

The first step in choosing a machine learning algorithm is to narrow down the desired task. There are two possibilties:

  • Predicting a Category
  • Not Predicting a Category

If you are predicting a category you now need to look at the labels of your data:

  • Labeled
  • Unlabeled

If you have labeled data then you need to follow a classification approach. If you have unlabeled data you need to follow a clustering approach.

If you are not predicting a category you will follow a regression approach.

Algorithms

Linear Regression:

  • Type: Supervised Learning, Regression Algorithm
  • Description: Linear Regression is a regression algorithm used to estimate real values, e.g. the price of a house, using a regression line of the form $Y=mX+b$ where all variables have continuous values.
  • When to use it: Linear Regression should be used when there is a linear relationship between the variables in the dataset. The linear regression algorithm fits a line of best fit to a dataset, so if you are sure that your data is correlated linearly then this is a good algorithm to use. For example, house prices are usually linearly correlated with square feet. If the house is larger then it generally costs more. Using linear regression makes sense in this case.
  • Pros: When the relationship betwen variables is linear the results are most always optimal. It is a very simple and computationally inexpensive algorithm to implement.
  • Cons: If your dataset has an outlier then this can skew the regression line and cause poor performance. Additionally, whenever a linear regression model is fit to a group of data, the range of the data should be carefully observed. Attempting to use a regression equation to predict values outside of this range is often inappropriate, and may yield incredibly wrong answers. Linear regression also only looks at linear relationships. If the data has a curve to it or is not exactly linear then this model is not going to produce great performance measurments like some others might.
